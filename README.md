- Run ```prepare_datasets.py``` (which calls py files in folder ```prepare_dataset```) to pre-process the dataset. 

- You may use ```prepare_datasets.py``` to pre-crop the 3D images if you find the training is slow.

- **\[IMPORTANT\] ```prepare_datasets.py``` generates a python dict (stored in ```dataset_info```) for each processed dataset. The dict contains file paths to each train and test files. All dataloading operations during training and testing depend on the dict. You should create your own dict. See dict samples in ```dataset_info```.**

## Train and test

Run ```train_voxDANN_1219.py``` to train the model on the ATAS dataset, and run other train_script to train the model on the corresponding dataset. They were implemented by PyTorch. You can easily adjust the hyperparameters.

For the corresponding dataset, the checkpoint can be evaluated using the corresponding sheet to obtain quantitative scores. Qualitative prediction results can also be generated by running `qualitive_visATAS.py`, which visualizes the predicted cellular instances.
